# Default Sound Pack

This folder contains the default sounds for Fahh.

- `fahh.mp3`: The original default sound.
- `fahhhard.mp3`: A harder-hitting sound for failures.
- `fartreverb.mp3`: A humorous reverb fart sound for failures.
- `fahhdeep.mp3`: A deep, resonant sound for failures.
- `fahhbroke.mp3`: A sound indicating something is broken.
- `ohshit.mp3`: A sound expressing surprise/frustration.
